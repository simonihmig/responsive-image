# @responsive-image/vue

Vue.js component to render responsive images, provided as locally processed images or loaded remotely from Image CDNs.

Part of the [ResponsiveImage](https://github.com/simonihmig/responsive-image) project.

## Compatibility

- Vue 3.5 or above

## Documentation

Visit [responsive-image.dev](https://responsive-image.dev) for our complete documentation site.
